# Мультиагентная система репликации кода по научной статье

## О проекте

Проект посвящён автоматизации начального этапа репликации AI-исследования по тексту научной статьи. Система принимает статью, извлекает из неё структурированное описание метода, экспериментального контура и зависимостей, сохраняет эти сведения во внешнем графе знаний и использует их для координации нескольких агентов, отвечающих за планирование, генерацию артефактов и проверку согласованности результата.

Цель системы — не заменить исследователя и не обещать полное воспроизведение результатов статьи, а получить трассируемый репликационный пакет, пригодный как стартовая точка дальнейшей ручной реализации.

## Что делает система

На вход подаётся текст статьи и связанные текстовые артефакты. На выходе система формирует:

- структурированное представление статьи;
- граф знаний по сущностям, связям, этапам метода и экспериментальным условиям;
- план реализации;
- черновые кодовые артефакты и файловый scaffold;
- перечень неявных допущений и пробелов в спецификации;
- результаты базовых автоматических проверок.

## Почему здесь нужна агентная архитектура

Задача состоит из нескольких слабо однородных этапов: извлечение фактов из статьи, нормализация и связывание сущностей, планирование, генерация кода, верификация и обработка ошибок. Для такой постановки естественна координация специализированных агентов, работающих поверх общего состояния и общей памяти. Внешний граф знаний используется как долговременное представление контекста и как источник трассируемости.

## Ограничения

Система не претендует на:

- полное автоматическое воспроизведение произвольной статьи;
- гарантированное достижение reported-метрик;
- автономное выполнение небезопасных действий;
- надёжную реконструкцию неописанных в статье инженерных деталей.
